# Freebot: Memory-Enabled AI Agent
A 24/7 Discord bot hosted on GCP with SQLite3 persistence and Google Search integration.

## Current Specs (2026)
- **Model:** Gemini 2.5 Flash-Lite
- **Database:** SQLite3 (`freebot_memory.db`)
- **Hosting:** Systemd service on Linux VM
- **Capabilities:** Memory (`remember`), Web Search, Image/Audio Perception.
